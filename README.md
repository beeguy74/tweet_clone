## HOW to start
You need python==3.9 to get started.
This is commands that you enter into the command line (linux or macOS):
```
$ python3 -m venv venv 
```
```
$ source venv/bin/activate
```
```
$ pip install pipenv
```
```
$ pipenv sync
```
```
$ python manage.py runserver
```
